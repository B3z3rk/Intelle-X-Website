About the Project

IntelleX is a smart document search engine designed to extract only the most relevant answers from your files.

It uses the BM25 ranking algorithm to locate top-matching documents and applies sentence-level filtering to isolate precise, query-aligned content.

For added clarity, it generates paraphrased answers using OpenAI's GPT-4 Turbo model.

Built with Python and Flask, IntelleX is secure, intuitive, and optimized for students and more to come.
